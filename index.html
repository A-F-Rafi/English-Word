Frequently Asked Questions

The difference between var, let, and const
var (Function and global-scoped, Hoisted, Can be Reassigned), let (Block-scoped, Can be Reassigned), const (Block-scoped, Cannot be Reassigned)
The difference between map(), forEach(), and filter()
map( )Creates a new array, Return Value , forEach() No Return Value , Iterates, filter() Condition Based , Creates a new array.
Explain arrow functions and how they are different from regular functions
Arrow functions are introduced in ES6, Arrow functions use =>. Regular functions use function words.
How JavaScript Promises work
A Promise in JavaScript is an object that represents the eventual completion (or failure) of an asynchronous operation. It allows you to handle asynchronous tasks in a cleaner way compared to callbacks.
How closures work in JavaScript
A closure in JavaScript is a function that "remembers" the variables from its outer scope even after the outer function has finished executing. This allows a function to retain access to variables from its parent function, even when it is executed later.
